# Central do Doce

Sistema web para gestão de confeitaria, criado para auxiliar no controle de ingredientes, receitas, custos fixos e precificação de produtos.

## Funcionalidades

- Login com opção de exibir ou ocultar senha.
- Dashboard com resumo dos dados cadastrados.
- Cadastro, edição e exclusão de ingredientes.
- Cadastro, edição e exclusão de custos fixos.
- Cadastro, edição, listagem e detalhamento de receitas.
- Cálculo automático de custo total, custo por unidade e preço sugerido.
- Busca de endereço por CEP.
- Alternância entre modo claro e modo escuro.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Flexbox
- Grid Layout

## Como Executar

Para executar o projeto localmente, abra o arquivo `index.html` em qualquer navegador moderno.

- `index.html`: estrutura das telas do sistema.
